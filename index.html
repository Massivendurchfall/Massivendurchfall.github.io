Python Basics – Visual Guide

Willkommen

Diese Seite erklärt Python so, dass auch absolute Anfänger ohne Vorkenntnisse verstehen, wie es funktioniert. Jeder Abschnitt enthält einfache Beispiele aus dem Alltag.

1. Erste Schritte

Python ist eine Sprache, die wie normale Sätze aussieht. Das macht sie besonders einfach.

Hallo Welt

Das erste Programm gibt einfach Text auf dem Bildschirm aus.

print("Hallo Welt")

Rechnen wie ein Taschenrechner

Python kann direkt einfache Rechnungen lösen.

5 + 3
10 - 2
4 * 5
20 / 4

2. Variablen

Eine Variable ist wie eine Box, in die man etwas legt und ihr einen Namen gibt.

name = "Lisa"
alter = 12
print("Ich heiße", name, "und bin", alter, "Jahre alt.")

Hier legen wir den Text "Lisa" und die Zahl 12 in zwei Boxen ab und nutzen sie im Satz.

3. Datentypen

Python kennt verschiedene Arten von Daten. Einfache Beispiele helfen dir, sie zu unterscheiden.

Zahlen

apfel = 3
banane = 2
obstkorb = apfel + banane
print("Im Korb sind", obstkorb, "Früchte.")

Text

gruß = "Hallo"
name = "Tom"
print(gruß + " " + name)

4. Entscheidungen

Manchmal soll das Programm nur etwas tun, wenn eine Bedingung erfüllt ist.

alter = 18
if alter >= 18:
    print("Du darfst Auto fahren.")
else:
    print("Du bist noch zu jung.")

5. Schleifen

Eine Schleife wiederholt etwas, bis es fertig ist.

for i in range(3):
    print("Das ist Runde", i+1)

Hier läuft die Schleife 3-mal und zeigt die aktuelle Runde an.

6. Funktionen

Eine Funktion ist wie ein Rezept: Einmal aufschreiben, beliebig oft nutzen.

def hallo(name):
    print("Hallo", name)

hallo("Mia")
hallo("Ben")

7. Listen

Listen sind Sammlungen, zum Beispiel eine Einkaufsliste.

einkauf = ["Milch", "Brot", "Äpfel"]
print(einkauf[0])  # Erstes Element

for ding in einkauf:
    print("Ich kaufe:", ding)

8. Module

Module sind Zusatzpakete mit neuen Funktionen. Beispiel: Mathematik.

import math
print(math.sqrt(16))

9. Texte (Strings)

Texte sind Zeichenketten. Man kann sie zusammenbauen und untersuchen.

Zusammenbauen

first = "Hallo"
last = "Welt"
msg = first + " " + last
print(msg)

f-Strings

name = "Mila"
age = 9
print(f"Ich bin {name} und {age} Jahre alt.")

Nützliche Methoden

text = "banana"
text.upper()
text.count("a")
text.replace("na","no")

Zerlegen

line = "Max;12;Berlin"
parts = line.split(";")
name, age, city = parts

Slicing

w = "Python"
w[0]
w[1:4]
w[-2:]

10. Wörterbücher (dict)

Ein Wörterbuch speichert Paare aus Schlüssel und Wert.

Erstellen und Nutzen

user = {"name":"Lea","age":10}
user["age"] = 11
print(user["name"])

Durchlaufen

for k, v in user.items():
    print(k, v)

11. Mengen und Tupel

Mengen (set)

a = {1,2,3}
b = {3,4}
a | b
a & b
list({1,1,2,3})

Tupel (tuple)

pos = (10, 20)
x, y = pos
coords = (1, 2, 3)

12. Wahr oder falsch (bool)

Programme treffen Entscheidungen mit Wahrheitswerten.

Vergleiche

7 > 3
5 == 5
"a" != "b"

Logik

True and False
True or False
not True

Truthiness

bool(0)
bool("")
bool([1])

13. Weitere Schleifenwerkzeuge

while, break, continue

n = 3
while n > 0:
    n -= 1
    if n == 1:
        continue
    if n == 0:
        break

range und enumerate

for i in range(3):
    print(i)

items = ["a","b","c"]
for idx, item in enumerate(items):
    print(idx, item)

zip

names = ["Max","Lea"]
ages = [9, 10]
for n, a in zip(names, ages):
    print(n, a)

14. Eingabe und Umwandeln

input

name = input("Dein Name: ")
print("Hallo", name)

Zahl aus Text

age_txt = "12"
age = int(age_txt)
print(age + 1)

15. Dateien lesen und schreiben

Schreiben

with open("notiz.txt","w") as f:
    f.write("Hallo")

Lesen

with open("notiz.txt") as f:
    data = f.read()
print(data)

16. Mini-Projekte

Temperatur-Umrechner

c = float(input("Celsius: "))
f = c * 9/5 + 32
print(f"Fahrenheit: {f}")

Zahlenraten

secret = 7
guess = int(input("Rate: "))
if guess == secret:
    print("Treffer")
else:
    print("Daneben")
