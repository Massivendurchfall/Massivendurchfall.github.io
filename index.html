Einrückung

Python strukturiert Blöcke über Leerzeichen. Üblich sind 4 Leerzeichen pro Ebene.

def greet(name):
    if name:
        print(f"Hallo {name}")
    else:
        print("Hallo Welt")

5. Funktionen

Parameter, Rückgaben, Standardwerte, Lambda, *args/**kwargs.

def area(w, h=1):
    return w * h

area(4, 2)
area(5)

def total(*values, tax=0):
    s = sum(values)
    return s * (1 + tax)

square = lambda x: x*x

6. Kontrollfluss

Bedingungen

    msg = "klein"

Schleifen

    print(i)

n = 3
while n:
    n -= 1

Listenabstraktion

[i*i for i in range(5) if i%2==0]

Fehlerbehandlung

try:
    10/0
