🎮 Nintendo Switch Emulation Guide

Play Tomodachi Life
on your PC

A clean, step-by-step guide to set up the Ryujinx emulator and get the game running.

⚙️ What You Need

📦 WinRAR
💻 Windows 10/11 (64-bit)
🎮 Ryujinx Emulator
🔑 Prod/Title Keys v22
💾 Firmware v22.0.0
🌍 Tomodachi Life (US NSP)
1
Download Everything First

📥 Required Downloads

Download all three files before doing anything else. You'll need WinRAR installed to extract them.

⬇ Ryujinx Emulator (.zip)
⬇ Firmware 22.0.0 (.zip)
⬇ Keys v22.0.0 (.zip)
⚠ WinRAR Required: Make sure WinRAR is installed on your PC before proceeding. You can get it free at winrar.com. It's needed to extract all the files.
2
Set Up Ryujinx

📂 Extract the Emulator

  • Right-click ryujinx-canary-1.3.272-win_x64.zip → Extract Here (with WinRAR).
  • Open the extracted folder and launch Ryujinx.exe.
  • After it opens, you can delete the original .zip — it's no longer needed.

🔕 Disable the Console Window

  • In Ryujinx, click Options in the top menu bar.
  • Find "Show Console" and uncheck it to hide the debug console window.
💡 Tip: This keeps Ryujinx clean and prevents an extra black window from appearing every time you play.
3
Install Keys

🔑 Extract & Install the Keys

  • Right-click ProdKeys.NET-v22.0.0.zip → Extract Here.
  • You'll get a prod.keys and/or title.keys file.
  • In Ryujinx, go to Actions → Install Keys → Install Keys (.KEYS).
  • Select the prod.keys (or title.keys) file from your extracted folder.
🖼 See the screenshots above — use Actions → Install Keys → Install Keys (.KEYS) from the menu.
4
Install Firmware

💾 Install the Firmware (No Extraction Needed!)

  • Do NOT extract Firmware.22.0.0.zip — keep it as a .zip file.
  • In Ryujinx, go to Actions → Install Firmware → Install Firmware (XCI or ZIP).
  • Select the Firmware.22.0.0.zip file directly.
  • Ryujinx will install it automatically.
🖼 Use Actions → Install Firmware → Install Firmware (XCI or ZIP) — exactly as shown in the screenshots.
💡 Note: Unlike the keys, you select the firmware ZIP directly without extracting it first.
5
Verify Everything Is Installed

✅ Check the Status Bar

Look at the bottom-right corner of Ryujinx. You should see something like this:

Firmware: 22.0.0  |  Keys: 22.0.0
✅ You're good if both Firmware 22.0.0 and Keys 22.0.0 are shown. If only one appears, repeat the relevant step above.
6
Add Your Games Folder

📁 Point Ryujinx to Your Games

  • Go to Options → Settings.
  • In the Interface tab, find "Game Directories".
  • Click Add and select the folder where you'll keep your game files (e.g., C:\Games\Switch\).
  • Optionally, add a second folder for DLCs/Updates in the "Autoload DLC/Updates Directories" section below.
🖼 This matches the Settings screenshot — the Interface tab shows Game Directories at the top.
7
Download & Install Tomodachi Life

🎮 Download the Game (2 Parts)

The game is split into two parts. You need both to extract it properly.

⬇ Tomodachi Life – Part 1 (.rar)
⬇ Tomodachi Life – Part 2 (.rar)
⚠ Important: Both parts must be in the same folder before extracting. Do not move one without the other.

📦 Extract the Game with WinRAR

  • Make sure both .part1.rar and .part2.rar are in the same folder.
  • Right-click .part1.rar → Extract Here (WinRAR will auto-detect Part 2).
  • You'll get an .nsp file — move it into your Games folder you set up in Step 6.
💡 Only right-click Part 1 — WinRAR will automatically use Part 2 during extraction.
🎉

You're All Set!

Ryujinx should now display Tomodachi Life in your game list. Double-click it to start playing!
